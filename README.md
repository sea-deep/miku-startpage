# Miku Startpage

A clean, fast, and minimalist custom homepage for your web browser. It's designed to keep you focused on what you need, completely free of ads and clutter.

![Miku Startpage](assets/screenshots/swappy-20260614-202400.png)

## Features

- **Sleek & Flat Design:** A pure dark mode layout with perfectly matching colors and smooth fade-in animations when the page loads.
- **Quick Search Engine Switching:** Click the icon inside the search bar to instantly switch between Google, DuckDuckGo, Bing, Startpage, Yandex, Brave, Ecosia, and Kagi. Your choice is automatically saved for your next visit.
- **Instant Keyboard Focus:** Just press the `Escape` key from anywhere on the page to instantly jump straight to the search bar and start typing. 
- **Numbered Bookmark Shortcuts:** Every website in your grid has a small number. Type that exact number into the search bar and hit `Enter` to instantly launch the website.
- **No Internet Required to Load:** It's built purely with HTML, CSS, and JavaScript. All icons and logic are saved right on your computer.

## How to Use

To use this, simply open the `index.html` file in your web browser. You can also set your browser's "Homepage" or "New Tab" setting to point to this file so it opens automatically every time!
